# libbstring

This is a package of Paul Hsieh's Better String Library, with makefile and
Visual Studio solution, for local usage (i.e. inclusion as a git submodule), to
avoid linking against the system supplied versions of the library.

This codebase is __not__ affiliated with Paul Hsieh in any way, shape, or form.

The original code and documentation may be found at:

* [Better String Library](http://bstring.sourceforge.net/)

Please support the official release.

Mike Steinert has produced an excellent fork of the Better String Library with a
whole suite of improvements (better build system, continuous integration, etc.);
it may be found in the [msteinert/bstring](https://github.com/msteinert/bstring)
repository.
